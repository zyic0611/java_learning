# 总结

划分基础与进阶范式 并不是依据在系统架构里的层级高低 而是它们的实现复杂度和历史演进节点。

cot react被称为基础的原因是 单线操作 没有引入复杂算法 并且都是纯提示词实现 

cot打开了内部推理的大门 react打开了外部交互的大门 所以被称为基础。

tot got都需要写外部的复杂代码。所以称为进阶。



## 层级结构

###  宏观调度层(规划):

采用 路由分发 或者 **规划执行** 模式。

不负责具体的任务 把一个目标拆解 输入原始需求 输出一系列子任务，

### 执行范式层

拿到子任务 决定如何完成 是执行者。

有 react tot got 



### 原子推理层

cot 是神经元 大模型最基础的推理能力 在tot react cot里的具体节点 本质上都是cot。



## 模型底座

除了cot思维链 还有standard 直觉回答 靠直觉回答 不进行思考 很容易出错 但是说明cot不是模型必备的 是复杂系统必备的
