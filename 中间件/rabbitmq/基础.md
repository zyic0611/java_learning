# 基础

生产者：生产数据、发送消息的程序

交换机：消息不会直接进队列 而是被发进交换机 会根据Binding规则 发给对应的队列

队列 存储消息的容器

消费者 监听队列消费消息的程序



Routing Key：生产者发消息给Exchange的时候 会在消息上贴一个标签

Bingding 绑定 Exchange会根据绑定信息 知道把Routing Key=..的消息发给哪个队列



Exchange的三大分发模式：

1. 直连模式 Routing Key必须和队列绑定的Key一样
2. 广播模式 不看RoutingKey 会把消息发给所有绑定到Exchange上的队列
3. 主题模式 使用通配符来匹配Routing Key
