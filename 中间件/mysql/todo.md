### 01_MySQL_基础与架构层

这个目录存放最基础的用法和 MySQL 的宏观执行逻辑。

- **01_MySQL整体架构** (Server层与存储引擎层、SQL执行流程、查询缓存、解析器、优化器)
- **02_数据类型与表设计** (整型、浮点、时间、字符串类型选择、三范式与反范式设计)
- **03_基础SQL查询与函数** (基础CRUD、聚合函数、字符串/时间/数学函数)
- **04_进阶查询语法** (多表JOIN原理及分类、子查询、CTE公用表表达式)
- **05_数据库对象** (视图 View、存储过程 Stored Procedure、触发器 Trigger、事件 Event)

### 02_MySQL_存储引擎与底层结构 (核心)

这个目录重点整理数据是如何在磁盘和内存中组织和存储的。

- **01_存储引擎对比** (InnoDB vs MyISAM、Memory、CSV引擎特性与适用场景)
- **02_InnoDB内存架构** (Buffer Pool缓冲池、Change Buffer写缓冲、Adaptive Hash Index自适应哈希索引、Log Buffer)
- **03_InnoDB磁盘架构** (表空间 Tablespace、段 Segment、区 Extent、页 Page 的层级关系)
- **04_数据页结构与行格式** (Page Directory、Compact/Dynamic行记录格式、行溢出机制)

### 03_MySQL_索引机制 (极度重要)

后端开发最核心的考点和日常工作痛点，这部分笔记需要极其详实。

- **01_索引底层数据结构** (B+Tree的演进与优势、Hash索引、为什么不用B-Tree或红黑树)
- **02_聚簇索引与非聚簇索引** (主键索引与二级索引、回表查询机制)
- **03_联合索引与最左前缀原则** (复合索引的B+树长什么样、最左匹配原理、索引下推 ICP)
- **04_索引设计与创建原则** (如何选择索引列、前缀索引、区分度计算)
- **05_Explain执行计划分析** (id、type、key、rows、Extra等字段的深度解析)
- **06_索引失效场景大全** (隐式类型转换、函数运算、左模糊匹配、OR条件等失效案例汇总)

### 04_MySQL_事务与并发控制 (极度重要)

涉及数据一致性和高并发处理的核心理论。

- **01_事务基础与ACID特性** (原子性、一致性、隔离性、持久性的实现基石)
- **02_并发事务带来的问题** (脏写、脏读、不可重复读、幻读)
- **03_SQL标准隔离级别** (RU、RC、RR、Serializable在MySQL中的具体表现)
- **04_MVCC多版本并发控制** (隐式字段、Undo Log版本链、Read View读视图的生成与匹配规则)

### 05_MySQL_锁机制 (核心难点)

排查线上死锁、理解并发冲突的必经之路。

- **01_全局锁与表级锁** (FTWRL、元数据锁 MDL、意向锁 Intention Locks、自增锁 AUTO-INC)
- **02_行级锁与算法** (记录锁 Record Lock、间隙锁 Gap Lock、临键锁 Next-Key Lock)
- **03_RR隔离级别下的加锁规则** (等值查询、范围查询的加锁退化逻辑，如何解决幻读)
- **04_死锁分析与排查** (死锁产生的四个条件、`show engine innodb status` 日志分析、死锁避免策略)
- **05_乐观锁与悲观锁实战** (版本号机制 CAS、`SELECT ... FOR UPDATE` 应用场景)

### 06_MySQL_三大核心日志体系 (底层基石)

了解 MySQL 如何保证数据不丢失以及如何进行集群同步。

- **01_Redo Log (重做日志)** (物理日志、WAL技术、两阶段提交、Crash-Safe能力、刷盘策略 `innodb_flush_log_at_trx_commit`)
- **02_Undo Log (回滚日志)** (逻辑日志、事务回滚实现、支撑MVCC)
- **03_Binlog (归档日志)** (Statement/Row/Mixed格式对比、主从复制的基石、刷盘策略 `sync_binlog`)
- **04_其他日志** (错误日志、通用查询日志、中继日志 Relay Log)
