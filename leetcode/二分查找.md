# 二分查找

为了不让你搞混，你可以直接记下这两个模板：

#### **门派 A：找 target（精确查找）**

- **循环条件**：`while (left <= right)`
- **缩减逻辑**：`left = mid + 1`, `right = mid - 1`
- **返回值**：在循环内部 `return mid`；循环外 `return -1`。
- **心法**：每一个数我都检查过了，没找到就是没找到。

#### **门派 B：找边界/极值（模糊查找）**

- **循环条件**：`while (left < right)`
- **缩减逻辑**：一边是 `mid + 1`，另一边是 `mid`（保留候选者）。
- **返回值**：循环结束时 `left` 和 `right` 一定重合，直接 `return nums[left]`。
- **心法**：我一直在把区间往中间挤，最后剩下的那个点就是真相。
