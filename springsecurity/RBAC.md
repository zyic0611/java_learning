# RBAC：

## perms 权限标识符

模块:资源:操作

## 类型

**`M` (Menu/Directory - 目录)：**

- 代表左侧导航栏的一级分类，点击通常只是展开子菜单，不会跳转具体页面。
- 比如：名为“医生管理”，但没有实际路径。

**`C` (Component/Page - 菜单/页面)：**

- 代表左侧导航栏的二级菜单，点击后会在浏览器打开一个实际的前端页面。
- 比如：名为“医生列表”，前端路由可能叫 `/doctor/list_page`。

**`F` (Function/Button/API - 按钮或接口)：**

- 代表页面上的一个具体动作（比如“搜索”按钮、“删除”按钮），或者纯后端的 API 接口。**这类资源不会显示在前端的左侧菜单栏里**，只用于权限控制。
